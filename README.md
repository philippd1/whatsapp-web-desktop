# WhatsApp-Web-Desktop

### what this is

- a simple wrapper of https://web.whatsapp.com/ written in electron webview

### why

- https://web.whatsapp.com/ loads fast
- WhatsApp for Desktop is buggy

so why not try and get the best out of the two? ^^
